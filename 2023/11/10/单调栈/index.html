2420. 找到所有好下标

  • 用单调栈记录当前位置之前的非递增/递减的元素个数
  • 遍历找出符合题意的位置
class Solution {
    public List<Integer> goodIndices(int[] nums, int k) {
        List<Integer> res = new ArrayList<>();
        int n = nums.length;
        int[] pre = new int[n];
        int[] suf = new int[n];
        pre[0]  = 1;
        suf[n-1] = 1;
        Stack<Integer> st = new Stack<>();
        for(int i = 0;i<n;i++){
            if(!st.isEmpty()&&nums[i] > st.peek()){
                st.clear();
            }
            st.push(nums[i]);
            pre[i] = st.size();
        }
        st.clear();
        for(int i = n-1;i>=0;i--){
            if(!st.isEmpty()&&nums[i] > st.peek()){
                st.clear();
            }
            st.push(nums[i]);
            suf[i] = st.size();
        }
        for(int i = k;i<n-k;i++){
            if(pre[i-1] >= k && suf[i+1] >= k)
                res.add(i);
        }
        return res;
    }
}

1124. 表现良好的最长时间段

  • 「劳累天数大于不劳累天数」等价于「劳累天数减去不劳累天数大于 」。

    那么把劳累的一天视作 1，不劳累的一天视作 −1，则问题变为：

    计算 nums 的最长子数组，其元素和大于 0。

  • 我们维护一个单调递减队列，作为结果可能的起点

  • 然后从后往前枚举结果可能的终点，就是最后的答案

 public int longestWPI(int[] hours) {
        int n = hours.length;
        int[] pre = new int[n+1];
        for(int i = 1;i<=n;i++){
            pre[i] = pre[i-1]+(hours[i-1]>8?1:-1);
        }
        int res = 0;
        Stack<Integer> stack = new Stack<>();
        stack.push(0);
        for(int i = 1;i<=n;i++){
            if(pre[i]<pre[stack.peek()]){
                stack.push(i);
            }
        }
        for(int i = n;i>0;i--){
            while(!stack.isEmpty() && pre[i]>pre[stack.peek()]){
                res = Math.max(res,i-stack.pop());
            }
        }
        return res;
    }

1673. 找出最具竞争力的子序列

  • 维护一个单调递减栈
  • 前提是当前栈内元素个数+剩余元素个数>k
class Solution {
    public int[] mostCompetitive(int[] nums, int k) {
        Stack<Integer> st = new Stack<>();
        int n = nums.length;
        for(int i = 0;i<n;i++){
            while(!st.isEmpty() && st.peek()>nums[i] && st.size()+n-i>k){
                st.pop();
            }
            if(st.size()<k){
                st.push(nums[i]);
            }
        }
        int[] res = new int[k];
        for(int i = k-1;i>=0;i--){
            res[i] = st.pop();
        }
        return res;
    }
}

1438. 绝对差不超过限制的最长连续子数组

  • 用两个队列，一个单调递增一个单调递减。这样就维护了区间的最大值和最小值。
  • 当最大值最小值的差大于limit时，移动指针，维护窗口内元素符合题意。
class Solution {
    public int longestSubarray(int[] nums, int limit) {
        Deque<Integer> queMax = new LinkedList<Integer>();
        Deque<Integer> queMin = new LinkedList<Integer>();
        int n = nums.length;
        int left = 0, right = 0;
        int ret = 0;
        while (right < n) {
            while (!queMax.isEmpty() && queMax.peekLast() < nums[right]) {
                queMax.pollLast();
            }
            while (!queMin.isEmpty() && queMin.peekLast() > nums[right]) {
                queMin.pollLast();
            }
            queMax.offerLast(nums[right]);
            queMin.offerLast(nums[right]);
            while (!queMax.isEmpty() && !queMin.isEmpty() && queMax.peekFirst() - queMin.peekFirst() > limit) {
                if (nums[left] == queMin.peekFirst()) {
                    queMin.pollFirst();
                }
                if (nums[left] == queMax.peekFirst()) {
                    queMax.pollFirst();
                }
                left++;
            }
            ret = Math.max(ret, right - left + 1);
            right++;
        }
        return ret;
    }
}

2398. 预算内的最多机器人数目

  • 滑动窗口+单调栈
public int maximumRobots(int[] chargeTimes, int[] runningCosts, long budget) {
        int n = chargeTimes.length;
        int res = 0;
        int left = 0;
        Deque<Integer> dq = new LinkedList<>();
        long[] preSum = new long[n+1];
        for(int i = 1;i<=n;i++){
            preSum[i] = preSum[i-1]+runningCosts[i-1];
        }
        for(int right = 0;right<n;right++){
            while(!dq.isEmpty() &&chargeTimes[right] >= chargeTimes[dq.peekLast()]){
                dq.pollLast();
            }
            dq.offerLast(right);
            while(!dq.isEmpty() && (preSum[right+1]-preSum[left])*(right-left+1) + chargeTimes[dq.peekFirst()]> budget){
                left++;
                while(!dq.isEmpty() && dq.peekFirst() < left)
                    dq.pollFirst();
            }
            res = Math.max(res,right-left+1);
        }

        return res;
    }

1856. 子数组最小乘积的最大值

  • 我们枚举每一个数作为最小值，然后向两边扩展，找到左右两边第一个比它小的数
  • 我们定义一个数作为最小值的区间是开区间（left,right），那么这个区间内的数都大于等于nums[i]
class Solution {
    public int maxSumMinProduct(int[] nums) {
        int n = nums.length;
        int[] left = new int[n];
        int[] right = new int[n];
        Deque<Integer> q = new LinkedList<>();
        Arrays.fill(right,n); 
        for(int i = 0;i<n;i++){
            while(!q.isEmpty() && nums[i] <= nums[q.peekLast()]){
                int cur = q.pollLast();
                right[cur] = i;
            }
            if(q.isEmpty())
                left[i] = -1;
            else
                left[i] = q.peekLast();
            q.addLast(i);
        }
        long[] sum = new long[n+1];
        for(int i = 1;i<=n;i++){
            sum[i] = sum[i-1]+nums[i-1];
        }
        long res = 0;
        for(int i = 0;i<n;i++){
            res = Math.max(res,(sum[right[i]]-sum[left[i]+1])*nums[i]);
        }
        return (int)(res%1000000007);
    }
}
